# portfolio_with_html
Basic HTML Portfolio
A simple, lightweight, and structured personal portfolio website built using strictly standard HTML (no external/internal CSS and no JavaScript).

Project Features
Strictly HTML: Standard tags (<h1>, <p>, <ul>, <li>, etc.) are used for structure and content presentation.
Lucida Font Stack: Applied a clean sans-serif typography (Lucida Sans Unicode, Lucida Grande, Lucida) globally using the native HTML <font> tag.
Centered Header Layout: Centered name title and navigation menu using the <center> tag.
Nested Projects List: Organized developer projects with sub-bullets containing descriptions, technical features, and links.
Clickable SVG Icons: Gmail, GitHub, and LinkedIn links are aligned side-by-side using local, scalable SVG assets (gmail.svg, github.svg, linkedin.svg) with standard horizontal spacing (hspace).
